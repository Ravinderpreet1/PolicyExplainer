# PolicyExplainer+

PolicyExplainer+ is a hybrid AI web application that helps users understand, simplify, and compare policy documents.

## Features
- Upload or paste policy text
- Explainable chatbot
- Risk and importance flags
- Highlighted policy answers
- Policy comparison
- Optional LLM explanations

## Tech Stack
- Python
- Streamlit
- OpenAI API
- PyPDF2

## Deployment
Deployed on Streamlit Community Cloud with secure API key management.
